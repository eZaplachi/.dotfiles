# .dotfiles

Personal Dotfiles managed with GNU-Stow

Contains install script for Nix package manager

don't forget to download git submodules!!

'''
git submodule init
git submodule update
'''

Run from home ('~') directory using:

'''
cd ~
./.dotfiles/install.sh
'''
